# Google Search Suggestions (React)

A React application that mimics Google search suggestions with real-time filtering.

## Tech Stack
- React
- Vite
- JavaScript
- CSS
- pnpm

## Features
- Controlled input
- Real-time search filtering
- Component-based architecture

## Setup
pnpm install
pnpm run dev
