# REACT Projects Repository

A collection of React projects showcasing different web development concepts and techniques.

## Projects

### 1. Mini Portfolio Example Layout
A modern, responsive React portfolio application that filters and displays projects by category.

**Features:**
- Tab-based project filtering (Static, Responsive, Dynamic)
- Responsive design with React 19
- Vite-powered development server
- Component-based architecture
- ESLint configuration for code quality

**Location:** `/MINIPORTFOLIOEXAMPLELAYOUT`

**Getting Started:**
```bash
cd MINIPORTFOLIOEXAMPLELAYOUT
npm install
npm run dev
```

Visit `http://localhost:3000` to see the application running.

**Available Commands:**
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Check code quality
- `npm run preview` - Preview production build

[View Full Documentation](./MINIPORTFOLIOEXAMPLELAYOUT/README.md)
